# 特注改造 出図業務改善

**株式会社スギヤス 技術部 開発グループ**

特注改造対応における出図業務改善に関する概要・試験運用の記録を残すリポジトリです

---

## 概要

原紙1枚の設計変更で全案件の出図ファイルを手修正している現状を改善する。  
原紙データ（.xdw）を図番単位で一元管理し、構成リストに基づく動的結合で常に最新の出図ファイルを生成する。

## 実施ステップ

| ステップ | 内容 |
|----------|------|
| Step 1 | 原紙データの自動リネーム（スキャン後の .xdw → 図番名） |
| Step 2 | 構成リストに基づく出図ファイルの自動生成 |

## リポジトリ内容

- `docs/` — 仕様書（図解付き）
- `tests/logs/` — 試験運用の記録
